How Glucora
Works

A closed-loop system that senses, predicts, validates, and acts — automatically, safely, and transparently.

1

Continuous Sensing

A CGM sensor measures interstitial glucose every 5 minutes and transmits readings via Bluetooth Low Energy to the ESP32 AI controller.

2

AI Prediction

A TinyML GRU model runs on-device to forecast glucose levels 30–60 minutes ahead, with confidence scores and risk classification.

3

Safety Validation

Multi-stage safety checks verify every dose recommendation against IOB tracking, patient care plan limits, and hypoglycemia prevention rules.

4

Insulin Delivery

Validated doses are delivered by a precision micro-pump with real-time occlusion monitoring and immediate failure detection.

5

Alerts & Reports

The mobile app surfaces real-time alerts, AI explainability, trend charts, and weekly health reports to patients, doctors, and guardians.

📡

CGM Sensor

FreeStyle Libre-compatible, BLE transmission, 5-min intervals

⚡

ESP32 Controller

Edge AI inference, safety logic, pump control

📱

Mobile App

Flutter, real-time dashboard, AI explainability

☁️

Cloud Backend

Supabase, secure storage, remote clinician access

AI Prediction Engine

The AI Engine
Behind Glucora

Three time-series forecasting architectures were evaluated and compared. GRU emerged as the optimal choice for real-time embedded deployment.

Trained on 22.6 million readings across 4 datasets from China, Spain, USA, and Egypt
8-bit integer quantization for ESP32 deployment with <500ms inference
Outputs predicted value, confidence score, and risk level (Low / Medium / High)
Anomaly detection monitors delivery command patterns for security threats
Adaptive pressure-based fault detection for real-time pump occlusion monitoring
GRU

Gated Recurrent Unit — Primary Model

Best-performing model. Efficient sequential learning with gating mechanism. Deployed on ESP32 via TensorFlow Lite.

⭐ Primary
LSTM

Long Short-Term Memory

Captures long-range temporal dependencies in glucose time series. Evaluated for 30–60 min forecasting accuracy.

Evaluated
HMM

Hidden Markov Model

Probabilistic model for glucose state transitions. Provides interpretable state-based predictions for clinical transparency.

Evaluated
RMSE

Evaluation Metric

Root Mean Square Error used to compare prediction accuracy across all three model architectures on test datasets.

Safety First

Built for
Medical-Grade Reliability

Every layer of Glucora is engineered with patient safety as the non-negotiable foundation. No dose is ever delivered without passing all checks.

⚖️

IOB Stacking Prevention

Real-time insulin-on-board tracking using exponential decay models prevents dangerous insulin accumulation.

🚫

Dose Cap Enforcement

Individualized maximum bolus limits from the patient's care plan are enforced regardless of the AI's recommendation.

🔍

Mechanical Fault Detection

Pressure profile analysis detects pump occlusions and tube disconnections in real time, halting delivery immediately.

🔐

Cyber Threat Detection

Behavioral anomaly scoring (0–1) flags suspicious delivery command patterns and can auto-suspend insulin delivery.
